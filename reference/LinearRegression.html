LinearRegression

linear_regression.LinearRegression(self, alpha, p, phi=1)

Treatment effect tests from Lindon et al. (2024).

Coefficients are calculated using the Recursive Least Squares algorithm.

See example.

Attributes

Name Type Description
beta np.ndarray Estimate of regression coefficients.
covariance np.ndarray Estimate of covariance matrix.
yty float Sum of squared response values.
Xty np.ndarray Sum of products of covariates and response.
